«Движение личного состава» - сервис предназначен для анализа передвижений аттестованного состава из службы в службу.

Имеются ввиду только переводы из одной службы в другую в УВД по Ивановской области в целом...

Не входят:

- передвижения из одного подразделения в другое без смены службы;

- принятые с «улицы»;

- прибывшие из других областей и ведомств.
